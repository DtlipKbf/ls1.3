HTML Tutorial

What is HTML?

Hypeter Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Web browsers receive HTML documents from a web server or from local storage and render the documents into multimedia web pages. HTML describes the stucture of a web page semantically and originally included cues for appearance of the document.

HTML elements are building blocks of HTML pages. With HTML constructs, images and other objects 
such as interactive forms may embedded into the rendered page. HTML provides a means to create 
structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, 
quotes and other item. HTML elements are delineated by tags, written using angle brackets. Tags such as 

<img/>
 and 
<input/>
 directly introduce content into the page. Others tags such as 
<p>
 surround and provide 
information about document text and may include other tags as sub-elements. Browsers do not display the 
HTML tags, but use them to interpret the content of the page.

Markup

HTML markup consists of several key components, including those cslled tags (and their attributes),
character-bases data types, character references and entity references. HTML tags most commonly come 
in pairs like 
<h1>
 and 
</h1>
, althought some represent empty elements and so are unpaired, for  example 
<img>
. 
The first tag such a pair is start tag, and the second is the end tag (they are also called opening tags and 
closing tags).

Another important companent id the HTML document decoration, which triggers standarts mode rendering.

The following is example of the classic "Hello, world!" program:

   <!DOCTYPE html>
    <html>
        <head>
            <title> This is tittle </title>
        </head>
        <body>
            <p> Hello world! /p>
        </body>
    </html>
